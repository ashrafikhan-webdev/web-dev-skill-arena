# Web Dev Skill Arena

An interactive browser-based quiz app for checking and improving core web development fundamentals.

The app starts with beginner-friendly questions to understand the learner's current level, then gradually moves into practical web development scenarios. It is designed for developers who want a more engaging way to test HTML, CSS, JavaScript, browser basics, APIs, Git, accessibility, and AI-era development habits.

## Features

- Level Check mode for basic fundamentals
- Fundamentals mode for broader beginner practice
- Next Step mode for practical scenario questions
- Timed questions with XP and streak scoring
- Instant explanations after every answer
- Final skill breakdown by topic
- Personalized training focus after each run
- Fully static app: no install, build step, or backend required

## How To Run

Open `index.html` directly in a browser.

```text
web-dev-skill-arena/index.html
```

## Learning Path

1. Start with **Level Check** to understand your current foundation.
2. Replay weak areas until the explanations feel obvious.
3. Move to **Fundamentals** for broader coverage.
4. Try **Next Step** once the basics feel comfortable.
5. Use the final roadmap to decide what to practice next.

## Why This Exists

Many developers can follow tutorials but still feel unsure about their fundamentals. This app makes the first diagnostic step more playful and less intimidating, while still giving clear feedback about what to improve next.
